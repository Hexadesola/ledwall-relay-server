Quick Start Guide

1. Connect to Controller

Select a COM port in the "Controller Connection" panel and connect. The status will change to "Connected" with a green indicator.

2. Control Lights

Use the ON/OFF buttons for each relay in the Front and Back Relay Control panels. Individual controls are at the top, or use "All Front Lights ON/OFF" buttons.

3. Control LED Display

Use the "Colour Selector" panel to choose a pattern (Solid, Blink, Wipe, Rainbow), speed, and color. Click "Apply to Selected" or "Broadcast All" to send to boards.

4. Master Control

The toggle switch in the Master Control panel controls ALL lights, devices, and LEDs at once. Use the indicator colors to see current state (green=ON, gray=OFF, amber=MIXED).

5. Auto Mode

Enable Auto Mode to run automatic light shows that cycle through colors and patterns. Use the "Change Interval" slider to control speed.

6. Preset Light Shows

Choose from 12 preset light shows like "Warm Sunset", "Ocean Flow", "Rainbow", etc. Click a show button to start, "Stop Show" to end.

Connection Troubleshooting

Problem: Cannot Connect

  • Check physical USB connection between computer and controller
  • Verify the controller is powered on
  • Try a different USB port on your computer

Problem: Port Not Found

  • Click "Refresh COM Ports" in the Tools menu
  • Check Device Manager to see if COM port is assigned
  • Close other programs that might be using the port

Problem: Connected but No Response

  • Verify all cables are securely connected
  • Check that relay equipment is powered
  • Try disconnecting and reconnecting

Finding COM Port

Open Windows Device Manager, expand "Ports (COM & LPT)", look for "USB Serial Device" or similar. Note the COM number (e.g., COM7).

Operator Tips

Quick Controls

  • Master Control: Use the toggle to turn everything on/off at once
  • All Lights ON/OFF: Controls both Front and Back lights
  • All Devices ON/OFF: Controls Camera Power, AC Socket, Moving Head, LED Display

LED Display

  • LED Display Power (Relay 28) must be ON for any LED patterns to show
  • Use "All LEDs ON" to quickly enable display with white color
  • Use "All LEDs OFF" to turn off all LED activity

Auto Mode

  • Auto Mode automatically cycles through light shows
  • Use "Change Interval" to set how long each show lasts (15 sec to 10 min)
  • Manual control overrides Auto Mode temporarily

Colour Presets

  • Use Colour Palette for quick access to preset colors
  • "Rainbow" button sets rainbow pattern with high speed
